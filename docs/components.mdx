## üìù TodoList Component

A comprehensive todo management system with filtering and priority features.

### Features

*   **Add Todos**: Create new tasks with priority levels
    
*   **Toggle Completion**: Mark tasks as complete/incomplete
    
*   **Delete Todos**: Remove unwanted tasks
    
*   **Filter System**: View all, active, or completed todos
    
*   **Priority Levels**: Low, Medium, High priority classification
    
*   **Statistics**: Real-time count of total, active, and completed tasks
    

### Usage

```tsx
import TodoList from './components/TodoList'

function App() {
  return <TodoList />
}

```

### Data Structure

```tsx
interface Todo {
  id: number
  text: string
  completed: boolean
  priority: 'low' | 'medium' | 'high'
}

```

### Key Functions

*   `addTodo()` - Creates new todo items
    
*   `toggleTodo(id)` - Toggles completion status
    
*   `deleteTodo(id)` - Removes todo from list

## üë§ UserCard Component

Displays user profile information fetched from an external API with navigation controls.

### Features

*   **API Integration**: Fetches data from JSONPlaceholder API
    
*   **User Navigation**: Previous/Next/Random user selection
    
*   **Loading States**: Shows loading spinner during data fetch
    
*   **Error Handling**: Displays error messages with retry functionality
    
*   **Responsive Design**: Adapts to different screen sizes
    

### Usage

```tsx
import UserCard from './components/UserCard'

function App() {
  return <UserCard />
}

```

### API Integration

```tsx
const fetchUsers = async () => {
  const response = await fetch('https://jsonplaceholder.typicode.com/users')
  const userData = await response.json()
  setUsers(userData)
}

```

### User Data Structure

interface User {
